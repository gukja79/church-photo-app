# 서울고 교실 사진

교회 자원봉사자가 모임 전·후 교실 상태를 사진으로 기록하고, 자동으로 Google Drive에 정리해주는 모바일 친화 웹앱입니다.

> 22개 장소(2층 교실 10개, 3층 교실 9개, 1층 화장실 3개)를 한 화면에서 선택하여
> 모임 전(Before) / 모임 후(After) 사진을 찍어 올리면, 날짜·장소별로 자동 분류되어 저장됩니다.

---

## 사용 기술

- **프론트엔드**: HTML / CSS / Vanilla JavaScript (프레임워크 없음, 가벼운 정적 앱)
- **백엔드**: Google Apps Script (Web App)
- **저장소**: Google Drive (공유 드라이브)
- **호스팅**: GitHub Pages
- **폰트**: Nanum Gothic
- **인증**: PIN 코드 + HMAC-SHA256 서명 토큰 (30일 만료)

---

## 주요 기능

- 🔐 **PIN 인증** — 4자리 PIN으로 간편 로그인. 한 번 입력하면 30일간 자동 로그인. PIN 변경 시 모든 토큰 자동 무효화.
- 📷 **사진 촬영 & 업로드** — 모바일 후면 카메라를 직접 호출. 1920px / JPEG 80%로 자동 리사이즈하여 업로드.
- 📂 **자동 정리** — `날짜/장소/사진.jpg` 형식으로 Google Drive에 자동 분류. 파일명도 정렬 친화적(`1_전_2층_1-14_093015_001.jpg`).
- 🛡️ **동시 업로드 안전** — Apps Script LockService로 직렬화하여 같은 초·같은 장소 중복 업로드 시에도 일련번호가 자동 충돌 회피.
- 🖼️ **오늘 사진 보기** — 장소 진입 시 오늘 업로드된 사진을 모임 전/후로 구분해 썸네일로 표시. 탭하면 전체화면 라이트박스(좌우 스와이프, 삭제 가능).
- 🌗 **다크/라이트 자동 전환** — 시스템 설정에 따라 자동.
- ✅ **업로드 진행 표시** — 사진별 상태(대기/업로드 중/완료/실패), 1회 자동 재시도, 실패 시 수동 재시도.
- 📊 **진행 현황** — 홈 카드에 장소별 Before/After 장수 + 색상 점으로 한눈에 표시.

## 자원봉사자용 사용법

### 처음 한 번만

1. 안내받은 앱 URL을 모바일(아이폰 사파리/안드로이드 크롬) 또는 컴퓨터 브라우저에서 엽니다.
2. **PIN을 입력**합니다 — 4자리 숫자. 한 번 성공하면 다음부터는 자동으로 로그인됩니다(약 30일).

> 💡 **홈 화면에 추가하기 (아이폰)**: 사파리 하단 공유 버튼 → "홈 화면에 추가". 앱처럼 빠르게 열 수 있습니다.

### 평소 사용 흐름

1. 홈 화면에서 본인이 담당한 장소 카드를 누릅니다 (예: `1-14`, `여자화장실`).
   - 카드 아래 작은 점이 파랗게 빛나면 모임 전 사진이 이미 올라와 있다는 뜻.
   - 초록색 점은 모임 후 사진이 있다는 뜻.
2. **"모임 전 사진 찍기"** 또는 **"모임 후 사진 찍기"** 큰 버튼을 누릅니다.
3. **카메라로 사진 찍기** 버튼 → 후면 카메라가 열립니다. 한 장 또는 여러 장 찍어도 됩니다.
4. 미리보기에서 잘못 찍은 사진은 우측 상단 ✕ 버튼으로 지울 수 있습니다.
5. 하단 **사진 N장 업로드** 버튼을 눌러 업로드합니다.
   - 사진별로 작은 스피너 → 초록 ✓로 변하면 완료.
   - 모두 완료되면 자동으로 장소 화면으로 돌아옵니다.

### 오늘 올린 사진 확인

장소 화면 아래쪽 **"오늘 업로드된 사진"** 섹션에서 확인할 수 있습니다.

- 모임 전 / 모임 후 그룹으로 나뉘어 썸네일이 보입니다.
- 썸네일을 탭하면 전체화면으로 크게 볼 수 있습니다.
- 전체화면에서 좌우로 스와이프하면 다음/이전 사진으로 넘어갑니다.
- 잘못 올린 사진은 전체화면 하단의 🗑 **삭제** 버튼으로 휴지통으로 보낼 수 있습니다 (30일 안에 복구 가능).
- 우측 상단 **↻** 버튼으로 새로고침할 수 있습니다 (다른 사람이 방금 올린 사진도 볼 수 있음).

### 다른 기기에서 다시 들어올 때

- PIN을 한 번 더 입력하면 됩니다.
- 우측 상단의 로그아웃 아이콘으로 명시적으로 로그아웃할 수도 있습니다.

## 폴더 구조

업로드된 사진은 Google Drive에 다음과 같이 자동 정리됩니다:

```
서울고 교실 사진/
└── 2026-04-29/
    ├── 01_2층_1-14/
    │   ├── 1_전_2층_1-14_093015_001.jpg
    │   ├── 1_전_2층_1-14_093015_002.jpg
    │   └── 2_후_2층_1-14_113045_001.jpg
    ├── 02_2층_1-13/
    ├── ...
    ├── 11_3층_2-5/
    ├── ...
    └── 20_1층_여자화장실/
```

- 날짜별 → 장소별로 한 폴더에 모임 전/후가 함께 들어갑니다.
- 폴더 이름의 두 자리 번호(`01`~`22`)는 앱의 표시 순서와 동일합니다 → Drive에서 한 화면에 22개 폴더가 정렬된 채로 보입니다.
- 파일명 첫 글자가 `1`(전) → `2`(후)이라 이름순 정렬 시 자연스럽게 시간 순으로 보입니다.

## 문의 / 문제 신고

- 앱 사용 중 문제가 있거나 PIN을 잊은 경우, 운영 담당자에게 문의해주세요.
- 잘못 올린 사진은 앱의 라이트박스(전체화면) 하단 삭제 버튼으로 직접 정리하실 수 있습니다.
